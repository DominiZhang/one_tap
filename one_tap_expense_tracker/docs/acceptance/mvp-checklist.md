## One Tap 记账 App MVP 验收清单

- [ ] 记账功能  
  - [ ] 打开 App 默认进入“快速记一笔”页面  
  - [ ] 输入金额 + 选择类别即可点击“保存”  
  - [ ] 连续记多笔时金额自动清空、类别保持最近一次选择  

- [ ] 明细功能  
  - [ ] 第二个 Tab 展示明细列表，按时间倒序排列  
  - [ ] 列表中显示日期分组头（YYYY-MM-DD）  
  - [ ] 左滑一条记录可删除，删除后列表立即更新  
  - [ ] 复制一条记录会生成新记录，金额/类别/账户/备注一致，时间为当前时间  

- [ ] 概览功能  
  - [ ] 第三个 Tab 展示本月支出总额  
  - [ ] 展示 Top 类别列表，金额与明细聚合结果一致  
  - [ ] 展示近 7 天趋势，每天金额与明细按日汇总一致（无记录的日期金额为 0）  

- [ ] 类别管理  
  - [ ] 可以新增类别，新增后可用于记账  
  - [ ] 可以重命名现有类别，明细和概览中展示新名称  
  - [ ] 可以归档类别，被归档类别不再出现在“启用中”列表  

- [ ] 账户管理  
  - [ ] 可以新增账户，新增后可在记账流程中选择  
  - [ ] 可以重命名现有账户，明细和概览中展示新名称  
  - [ ] 可以归档账户，被归档账户不再出现在“启用中”列表  

- [ ] 备份与导出  
  - [ ] JSON 导出包含 `schemaVersion`、`categories`、`accounts`、`transactions` 字段  
  - [ ] JSON 导入可完成一次 round-trip：导出 -> 删除本地数据 -> 导入 -> 数据一致  
  - [ ] CSV 导出包含表头：`date,type,amount,category,account,note`，并能在表格工具中正常打开  

- [ ] 自动化与质量  
  - [ ] `flutter test` 全部通过  
  - [ ] `flutter analyze` 无问题
